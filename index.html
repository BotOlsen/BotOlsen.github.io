About

New college graduate in computer science with honors from California State University, Sacramento. I have
a passion for all things space related in topics ranging from launching rockets to potential GIS of other planets.

Past projects I have worked on include a multitasking operating system written in C, a CPU designed in Verilog, a
game written in Java using the codenameone framework, and an iOS wellness application.

Currently learning about signal processing and general electrical engineering concepts in order to build a radio
telescope.
