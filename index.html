cosmOS

Description

This operating system, cosmOS, is a multitasking operating system written using SPEDE (System Programmers Educational Development Environment). This environment is a virtual machine that mimics hardware for the OS to utilize and standardizes implementation for various physical devices.

The run loop for cosmOS starts by performing a context switch which will save the current processes trapframe. Then the interrupt handler will run on whatever interrupt triggered the context switch. Once the interrupt is handled via itâ€™s interrupt service routine, the OS determines if there has been a keyboard key press by the user. These commands can create processes, exit processes, display kernel stats, and many more. After it has checked for user input, it will run the scheduler with a round robin implementation. After the scheduling, another context switch is performed, and the next process takes over the CPU.
